# jxb
Joomla Extension Building Action

## Inputs

### `build-file`

**Required** The name of the file to use as a guide, for building the extension. For available syntax, please read below. Default `"build.jxb"`.

## Example usage

uses: actions/jxb@v1
with:
  build-file: 'build.jxb'
